Privacy Shield

Privacy Shield is a simple Google Chrome Extension which detects and block third-party trackers. To do this task, Privacy Shield uses two lists.

  • Disconnect.me list of trackers
  • Privacy Badger yellow list

When Privacy Shield intercepts a request to a third party-domain, it analyzes it and tags it.

  • Blocked
    The domain request was recognized as a tracker and it was blocked.
  • Cookies
    The domain request was found in yellow list. This list allows requests to third-party domains only for functionality purposes, so the Cookie, Referer and Set-Cookie headers are deleted before sending the request or receiving the response.
  • Allowed
    The domain request were analyzed and it was allowed.

Privacy Shield permits to disable the extension in some web sites and offers an option page for managing it.
